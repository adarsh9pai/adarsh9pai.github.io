Some cool things I did

Clime | Digital Emergency Assistant

MLH TamuHack. January, 2018

  • Clime is a Digital Emergency Assistant built using Node.Js(Express) and the Facebook Messenger Platform. The chatbot can help you get real-time weather updates, find ATMs, Hospitals and Shelters near you at the time of disaster and will also guide you to help and donate! It was written at TamuHack and took about 22 hours in total to complete.
  • Check it out on Github.


MavScript

MLH HackUTA. October, 2017

  • Mavscript is an UT Arlington themed esoteric programming language. It was written at HackUTA and took about 17 hours in total to complete. Given the time constraints, it lacks the utility of a full programming language, but it's still capable of solving FIZZBUZZ and other fun algorithms!
  • It is dynamically typed and dependency free.
  • Check it out on Github.
